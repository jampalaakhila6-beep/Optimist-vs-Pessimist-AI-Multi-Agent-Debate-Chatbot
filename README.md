# 😂 Optimist vs Pessimist AI

A fun AI application built using Streamlit, AutoGen, and Google Gemini API.

## Features

- 🤖 Two AI agents debate your situation.
- 😎 Optimistic Friend
- 😒 Pessimistic Friend
- 💬 English + Telugu slang
- ⚡ Built with AutoGen

## Technologies

- Python
- Streamlit
- AutoGen
- Gemini API
- OpenAI Compatible SDK

## Installation

pip install -r requirements.txt

streamlit run app.py
